# Marzban Rules

Кастомные правила для Marzban (Clash).

## Структура

```
marzban-rules/
├── README.md
└── clash/
    └── custom.yml
```

## Использование

1. Подключи репозиторий в Marzban через переменные окружения
2. Правила загрузятся автоматически при деплое

## Правила

- 🇷🇺 Российские сайты → DIRECT
- 👾 Discord → отдельная группа
- 🤖 AI (OpenAI, ChatGPT) → отдельная группа
- 🚀 Остальное → прокси
